7.位运算符

位运算符都是对二进制数字进行操作的！继续看图：

按位与运算：

>>> a = 2      
>>> b = 3  
>>> a & b  
2

解析：

将a，b转成2进制进行计算，数列操作，个位上0&1=0，十位上1&1= 1。

按位或运算：

>>> a = 2      
>>> b = 3  
>>> a | b  
3

解析：

将a，b转成2进制进行计算，数列操作，个位上0|1=1，十位上1|1= 1。
